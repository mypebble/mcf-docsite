Getting to know Arro

Arro software is easy and intuitive to use. So what does it look like and how do you get started using it? When you first log on to Arro, if you have more than one school, you'll choose the school from the list of school Names. The Get Started - Choose a Project screen is displayed

The Arro interface is split into the following sections:

Header

The header identifies the school and the logged on user. The Amount Raised shows you how much you have raised for the project so far.

Breadcrumbs

Additionally, once you select a module and drill-down into the screens, a breadcrumbA way of navigating around a system. A trail of breadcrumbs can lead you back to where you came from easily and provide short-cuts to different parts of the system. trail appears across the top of the screen to help you keep track of where you are in Arro.

In some cases you can click onto a level and go there directly. For example, clicking on News articles in this breadcrumb trail displays the list of News articles.

Breadcrumbs offer another way of navigating around Arro

Navigation bar

The Navigation Bar shows all the modules available in ArroArro and you can select them from here. In addition the Navigation Bar has a Help & Support option at the foot which opens the Support website for Arro.

Work area

This is where you work within each module. The example shows the list of Your Projects in the Projects module in the work area.

Arro makes use of many usual features such as buttons to cause actions, tabbed menus, data entry forms to record data, fields that are hyperlinked for drill-down, and so on.

Action buttons

Buttons let you perform actions. For example, to view a list of your projects, you click Your Projects on the Get started - Choose a project screen.

Tabbed menus

Tabs are used in the Your projects screen in the Projects module to let you see lists of active and inactive projects.

These let you choose different views in the work area.

Hyperlink drill-downs

Where text is hyperlinked, you can click the link to drill-down to more detail.

Data Entry forms

These let you enter the data you need, for example, enter contact details.

Data entry forms include different types of fields. The New Contact form contains:

Selection field

You select from a drop-down of available options, for example the Contact Type field.

Data Entry field

You enter free text in these, for example, the First Name field.

Check box

You click to select and click again to clear, for example, the Gift Aid check box.

Elsewhere in Arro you may encounter the following types of field in data forms:

Date field

You select the date from the drop-down calendar, for example, the First Email field when sending a campaign.

Filter fields

A filter works to shorten a lengthy list of items. Filtering can be done several ways. In this example from the Grants module,

you can:

  • Toggle the display of grants listed between grants that your school is eligible for and all grants.
  • Enter free text in the Search field. This will shorten the list to show only those grants that include the free text which you have entered.
  • Select from a drop-down list of filters - Project Type, Grant Type, Complexity or Status. This will shorten the list to show only those which fulfil the criteria.
  • Click at the right-hand side of the Deadline column, to view grants you have previously saved as favourites.

    • For more information on saving grants, see Grants.

To move between fields in data entry forms, either use the mouse to click in the relevant field or use the TAB key to move from field to field.

Scrolling in drop-downs - how to do it

You can scroll in drop-downs either using the scroll bar in the drop-down or using the mouse scroll wheel when the mouse cursor is placed on the drop-down list.

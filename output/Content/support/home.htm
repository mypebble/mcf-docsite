Mission Statement

 

This is where you will find support on using Fund Manager - Pebble's Financial Reporting software - which is designed to help you keep a record of every penny you collect, MISapp - an application which runs alongside SIMS.net daily to import and update names within Fund Manager - and Joinos for Parents - online payment software, which helps schools manage the income they receive from parents for trips, uniform sales, dinner money, etc.

We provide a mix of User Guide PDFs and support article FAQs and you can use the Search facility to search the Support library for what you need. We want you to find the information that helps you so if you can't find what you are looking for here, please let us know.

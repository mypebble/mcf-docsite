MISapp Preparation Instructions

MISapp is a small application that runs alongside SIMS.net. It runs daily to import new and update existing names within Fund Manager. Some fields are mandatory as part of the import process however you have the option of excluding all or some of the following information:

  • UPN Is the Unique Pupil Number allocated to each pupil (This option only applies to new customers)
  • Contacts Are the parent/guardian contact details for each pupil
  • Groups Are the groups that a pupil is linked to i.e. free school meals
  • Staff Are the staff members held in your MIS system
  • Attendance Is the pupil attendance records

Note: If you would prefer to exclude any of the items listed above from being imported into Fund Manager you must notify a member of the support team before you run MISapp.

  1. SIMS.net Preparation

    2. You MUST have completed the SIMS.net preparation stages before installing MISapp. Follow the instructions below to ensure you have the correct permissions and access to SIMS.net, which is a requirement before you install the MISapp.

    You MUST be a member of the following groups - Admin Assistant, Class Teacher, Personnel Assistant and Third Party Reporting in order to use MISapp. To check if you are a member of these groups go to menu option:

    Focus > System Manager > Manage Users

    Search for your ID and check the groups you belong to in Section 3. If you do not have all these groups please contact your System Manager and ask to be added.

    Alternatively, you could create a System or Admin id which only uses these groups instead.

  2. Fund Manager Preparation

    3. Before installing MISapp, mark any pupils and staff that have left the school as “Out of Use”. For more information on how to manage names in Fund Manager please read Managing Names.

    Note: MISapp does not automatically mark school leavers out of use.

    When you have completed the preparations above download the MISapp Installation Instructions for the installation instructions for MISapp. The link to download MISapp is also in the email you were sent.

  3. Contact Pebble Support

    4. Once you have installed MISapp please call Pebble Support on 0845 310 1788 Option 1 and we will provide you with your “Password” which is required during the setup of MISapp.

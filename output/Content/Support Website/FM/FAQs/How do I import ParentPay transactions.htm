How do I import ParentPay transactions?

Solution: Create an import file in ParentPay and upload to Fund Manger

Within ParentPay go to Reports > Payments

  • choose service or all
  • select dates
  • click on advanced options
  • select Payment status: Processed
  • select Payment method: ParentPay online or Paypoint
  • open in excel

    • Note: In order not to import duplicate transactions always select at least the day before the current date as the “To date”. Choosing Processed will ensure only payments that have settled to a bank account will be included in the report

Method:

Note: Cash and cheque transactions should not be on this report, if you have filtered in ParentPay for online only transactions they should not appear- if they do you must delete these lines or start again

Remove all Negative transactions before importing - these will need to be entered manually after the import.

Adjust the Excel spreadsheet to have the following headings only:

  • Service name - modify to Item Title
  • Forename
  • Surname
  • Amount
  • Processed date – modify to Date

VERY IMPORTANT - delete the top blank line so your file will import

save the file to your chosen folder making sure to choose the file type CSV

Within Fund Manager go to Transactions > Import ePayments

  • choose file
  • Upload file
  • choose account
  • choose settlement account
  • work through payments matching name with name

Note: If the name or account does not appear then you may need to create a new name or account via Manage Names or Manage Accounts in Fund Manager and return to this screen to complete the import process (View Pending Import tab)

  • once you have matched the transactions click the Import button to complete

Note: You will not be allowed to import any transactions that you have not matched or if you have not chosen a Settlement Account to place these in

Once you have imported your transactions you will be able to transfer the total transaction amount from your statement from the settlement account to your chosen bank account by doing the following;

Bank Transfers to Reconcile

Go to Bank & Cash

  • Transfer between Bank Accounts

    • Transfer from > Settlement Account

    Transfer to > Chosen Bank Account

  • Enter amount
  • Enter date
  • Transfer

You can then Record a Payment for the debited fees on your bank account statement to balance (negative amounts).

Joinos Messages

You can create and send Joinos messages and sms text alerts to parents and carers of pupils. Telephone numbers are added in Fund Manager > Administration > Manage Names.

You can send attachments via Joinos if the documents have been uploaded using Fund Manager > Offers > Upload Document.

To send an email message

  1. Open Fund Manager > Messaging > Joinos New Message.
  2. Enter the Title and Body of the message.
  3. Select any listed attachments to attach to the message within Joinos.

    4. For more information see, To upload documents to attach to offers.

    Attachments are not attached to the email notifications but can be accessed by parents within their Joinos account.

  4. Select the names to send the message to in the Contacts list.
  5. Click Send.
  6. Parents receive a message in Joinos and an email notification.

To send an SMS alert

  1. Open Fund Manager > Messaging > Joinos New Message.
  2. Enter the Title and Body of the message.
  3. Select any listed attachments to send with the message.

    4. For more information see, To upload documents to attach to offers.

    Attachments are not attached to the SMS alerts but can be accessed by parents within their Joinos account.

  4. Select the names to send the message to in the Contacts list.
  5. Select Send selected contacts a message alert via SMS.
  6. Click Send.

To view message history

  1. Open Fund Manager > Messaging > Joinos History to display a list of messages that have been sent.
  2. Click the message Title to see the details of the message that was sent.
  3. Check the Read column to check if a recipient has read the message.

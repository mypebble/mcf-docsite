VAT 100

About the fields Box

The VAT 100 report displays nine boxes you are required to fill when completing the HMRC VAT Return.

You can bet a breakdown of the entries for a box by clicking the hyperlinked box number.

 Detail

The description of the VAT box.

Period Dates Amounts

The amounts totalled for each box during the period since the last report. Fund Manager totals all VAT transactions since the last report, including any that weren’t included on a previous VAT 100 report.

 Print

Click this to print a VAT Return report.

 Close VAT Period

Click this to close the VAT period.

 Abandon

Click this if you are not ready to close the VAT period. It is not available once you have closed the VAT period.

 Finished

This is available once you have closed the VAT period.

  • Click Finished to see all historical VAT Return reports.

Use this screen to preview and print VAT Return totals, close the current VAT period if it is still open, print a final VAT Return report or print the details of a historical VAT Return report.

You can cancel and amend transactions that have not yet been included in the final VAT Return report for the period. Provided you haven’t closed the VAT period you will be able to cancel or amend the transactions. Once you have closed the VAT period, you won’t be able to do this. Once you have closed the VAT period you also won’t be able to cancel or amend the final report for the period either.

How do I ...

Print the VAT Return as a preview report?

You can print a preview report for the period at any time until you close the period by clicking Close VAT period.

  • Click Print. A prompt tells you your report is being prepared. Once it has been prepared it is displayed as a PDF and you can then print the PDF file.
 Close a VAT Period?
  1. Click Close VAT Period.

    2. The following pop-up screen appears:

    Once you confirm on this screen that you want to close the period, you cannot reverse the action.

  2. If you are sure that you want to continue, click Close VAT 100 Period.

    3. The VAT 100 Period is closed and you can submit your VAT 100 report to the HMRC.

 Print the VAT Return as a final report?
  • Ensure that you close the period first by clicking Close VAT Period.
  • Then click Print.
 Print a historical VAT Return report
  • Click Print.

Period Beginning Date

The start date of the period.

 Contact Information

A contact address for the organisation, including two address lines, town, county and post code and the organisation’s unique reference number (URN).

This URN is the school’s own organisation number. This isn’t currently used by Pebble as the Post Code is used to access the organisation instead. There is a URN in Sage for Education but this is a different URN.

Use this screen to add and update organisation details.

How do I ...

Amend organisation details?
  1. Enter the organisation Name and select In Use to ensure you can select the organisation on the system.
  2. Select Messaging if the organisation is going to use the Messaging options.
  3. Select Sage if the organisation is integrating with Sage for Education.
  4. Select the Subscription Type.
  5. Select the date the Subscription Expires.
  6. Select the ePayment back-end integration product. If you select ParentPay the organisation will have a ParentPay menu, if you select sQuid it will have a sQuid menu and if you select fund tracker it will have an Offers tab.
  7. Select the Period Beginning Date.
  8. Enter the organisation Address and the organisation unique reference number (URN) if it has one.
  9. Click Save.
 Add a new organisation?
  • From Administration > Manage Organisations, click Create New Organisation.
 Remove organisation details?
  • Clear the In Use check box and click Save.
 Download the organisation details?

This is only available once the organisation has had data entered.

  • Click Download organisation details.

A zipped file with the organisation details is downloaded to your computer. The details include bank accounts, names, name groups, payment transactions, receipt transactions, user account transfers, user accounts, user accounts groups, and so on.

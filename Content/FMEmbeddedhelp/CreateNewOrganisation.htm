About the fields Organisation

The organisation’s name.

 In Use

Selected indicates that the organisation is in use.

 Contact Details

A contact name and address for the organisation, including post code, phone number and email address. You can add additional contact names by clicking the plus sign alongside Post Code. You can enter additional phone numbers and email addresses by clicking the plus sign alongside Phone Number and Email Address.

Use this screen to add and update organisation details, for example your school’s suppliers.

How do I ...

Add a new organisation?
  1. Enter the Organisation name and select In Use to ensure you can select the organisation on the system.
  2. Enter a Contact Name and Address for the organisation, including Post Code.
  3. Enter a Phone Number and Email Address for the contact.
  4. If required, click the plus icons alongside Post Code, Phone Number and Email Address in order to display more fields for the information you want to enter.
  5. Click Save.
 Amend organisation details?
  • From Administration > Manage Names, select the organisation name to display the Update Organisation screen.
 Remove organisation details?
  • From Administration > Manage Names, select the organisation name to display the Update Organisation screen.
  • Clear the In Use check box and click Save.

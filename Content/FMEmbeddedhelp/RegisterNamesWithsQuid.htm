sQuid Register Names with sQuid

About the fields List of Names

A list of names, showing transaction: Name, Class, Year, Reg and Groups.

 Select all filtered

If selected, all names displayed as a result of the Filter you have applied, are selected. If you don’t enter anything in the Filter box, then all the names displayed are selected.

Use this screen to register pupils who are not already set up on sQuid.

If you are an IMPACT school and receive your card numbers from a catering company, you must talk to us about your file before attempting to register those pupils.

How do I ...

Find Names?
  • Enter the Name you want to find (if you know it) in the Filter box.
  • Use the Filter to shorten the list of names that is displayed, by entering a character or characters you think are in the name.
 Register pupil names?
  • Select the pupil names you want to register and click Register Names.

    • Use the Filter box to shorten the list to find the names you want and then select the Select all filtered checkbox.

    Information is automatically sent to sQuid and when sQuid completes, you can see the names, using the View Registerd Names tab.

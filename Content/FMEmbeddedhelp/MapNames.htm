Map Names

About the fields List of Names

A list of names is displayed, showing: Name, Year, Registration group and a drop-down to Match Contact..

Names usually map automatically in Fund Manager from MISapp. Sometimes names are not automatically mapped. This is usually because they aren’t held in the MIS system with a permanent UPN. Use this screen to match any unmapped names.

To match unmapped names or import new ones from ParentPay:

  1. Select a Name in the list of names displayed on the left. Then select the name to match it to, from the Match Contact drop-down list on the right.
    2. If no matching name is displayed because this is a new name, select the box on the left of the new name.

    Please be aware of pupils or staff with the same name when matching names. If you are unsure, abandon the matching process and go to Administration > Manage Names to check.

  2. Click Import Selected.

    3. If you have 100 or more new names to select, you can select Import 100 and Fund Manager will import the first 100 unmatched names automatically.

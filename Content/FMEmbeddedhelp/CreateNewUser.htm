How do I ...

Add a new user?
  • Enter the user Email Address, First Name and Last Name,select In Useand click Save.
 Amend a user’s details?
  • Use Administration > Manage Users, select the user name to display the Update User screen.
 Remove user details?
  • Use Administration > Manage Users, select the user name to display the Update User screen.
  • Clear the In Use check box and click Save.

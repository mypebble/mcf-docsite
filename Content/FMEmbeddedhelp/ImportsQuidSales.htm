How do I ...

Import transactions?
  1. Select the Settlement Account for the money to be imported into.
  2. Select the transactions to be imported.
  3. Click Import.

    4. Once imported, the transactions show in Fund Manager against the correct user accounts and names. The money is added to the sQuid Settlement Account.

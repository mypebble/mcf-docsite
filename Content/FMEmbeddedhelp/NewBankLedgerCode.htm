New Bank Ledger Code

About the fields Code

The ledger code which is used to group transactions for a particular purpose or the bank ledger code which is used to match the bank in Fund Manager to the bank code in an external financial system.

 Name

The name of the code.

The VAT and Cash in Hand ledger codes are set up in Fund Manager by default and you can’t amend the name of these codes.

 In use

The In use flag only applies to ledger codes you have created yourself. It doesn’t apply to the VAT or Cash in Hand codes or to bank ledger codes.

The flag indicates whether the ledger code is in use or not: selected, the code is in use; clear, it is not in use. If a code is not in use then you won’t be able to select it for transactions.

 Bank account

The name of the bank account, selectable from the drop-down list of available bank accounts.

You can only select a bank account that has already been created using Administration > Manage Bank Accounts.

How do I ...

Create a ledger code
  1. Enter the Code and the Name of the account.
  2. To use the code, select In use.
  3. Click Save.
 Create a bank ledger code
  1. Enter the Code and select the Bank account.
  2. Click Save.
 Amend a ledger code
  • From Administration > Manage Ledger Codes, click the hyperlinked Code or Description to open the Update Ledger Codes screen and follow the instructions there.
 Amend a bank ledger code
  • From Administration > Manage Ledger Codes, click the hyperlinked Bank Ledger Code to open the Update Ledger Codes screen and follow the instructions there.
 Mark a ledger code in use

Marking a ledger code as in use means that you can use the ledger code and apply it to transactions.

  1. Select the In use flag.
  2. Click Save.
 Mark a ledger code out of use

Marking a ledger code as out of use means that you can no longer use the ledger code and apply it to transactions.

  1. Clear the In use flag.
  2. Click Save.

    3. You won’t be able to use the ledger code

Joinos Offer

About the fields Account

Select the Account for the offer from the drop-down list of available accounts. You can use the Filter box to enter the Account code or a few characters at the start of the account code to help you select the correct code quickly.

 VAT Code

Select the VAT Code applicable to the offer.

Settlement Account

Select the settlement account for the offer when parents are making online payments

 Title

Enter the offer Title which should give a succinct explanation of the offer, making it easily identifiable, e.g. Year 12 History Trip - Hadrian’s Wall.

 Description

Enter a description e.g. Total cost of trip is £300. This can be paid altogether or in instalments of £50.

Let parents know exactly what you want them to do and how they can pay for the offer.

 Gift Aid

Selected means that Gift Aid can be collected on the offer.

 Amount

If the offer is a set amount, enter the amount of the offer.

 Unspecified

If the offer is open to multiple payments, or the parent/carer decides how much to pay, select this.

If you want parents to see what they have paid so far you need to select the Share Balances box in Administration > Manage Accounts.

 Minimum

To ensure parents and carers can’t pay below an amount, enter the minimum amount here.

 Consent Required

Selected means parents and carers must give online consent for the offer.

 Start Date

The date the offer becomes visible to parents/carers.

 End Date

The date the offer is removed and parents and carers can no longer see it.

 Recurring

Selected means that parents can pay more than once.

Min

The minimum number of items which can be purchased.

 Max

The maximum number of items that can be purchased in one transaction.

 Items Available

The number of items available in the offer.

Unlimited

You have an inexhaustible supply of items.

 Attachments

If you want to add attachments e.g. the trip itinerary or cost breakdown, you can add these to the offer and they will be displayed on the offer screen for parents/carers to view.

We recommend uploading PDF documents rather than Office files.

Do not upload any documents containing sensitive or private information.

 Names

Choose the names to add to the offer, by selecting names from the list of pupil names displayed. To help you shorten the list you can use the Filter.

How do I ...

Create an offer?
  1. Select Account.
  2. Select VAT code if applicable.
  3. Select Settlement Account.
  4. Enter the offer Title and Description.
  5. Select Gift Aid if you want to claim gift aid on the offer.
  6. Select Consent Required if you want parents to give online consent.
  7. Enter the Amount.
  8. Select Unspecified if the offer is open to multiple payments.
  9. Enter a Minimum amount.
  10. Enter the offer dates.
  11. Select Recurring if required.
  12. Choose the offer quantities - Minimum, Maximum and Items Available.
  13. Select Unlimited if necessary.
  14. Add attachments to the offer if required.
  15. Select the names you want to make the offer available to.
  16. To save and post the offer, click Post Offer. To save the offer and work on it again later, click Save without Posting.

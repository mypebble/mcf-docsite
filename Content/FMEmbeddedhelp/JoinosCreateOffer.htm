Joinos Offer

About the fields Account

Select the Account for the offer from the drop-down list of available accounts. You can use the Filter box to enter the Account code or a few characters at the start of the account code to help you select the correct code quickly.

 VAT Code

Select the VAT Code applicable to the offer.

Ledger

Select the Ledger code applicable to the offer.

 Settlement Account

Select the settlement account for the offer when parents are making online payments

 Title

Enter the offer Title which should give a succinct explanation of the offer, making it easily identifiable, e.g. Year 12 History Trip - Hadrian’s Wall.

 Description

Enter a description e.g. Total cost of trip is £300. This can be paid altogether or in instalments of £50.

Let parents know exactly what you want them to do and how they can pay for the offer.

 Icon

You don’t have to associate an icon with the offer, but icons can help parents easily identify the types of offer you have available.

Click Choose Icon to associate an icon with the offer. A number of icons are available to choose. Choose the icon that is appropriate for the offer e.g. an icon showing a bus or a plane indicates travel.

 Gift Aid

Selected means that Gift Aid can be collected on the offer.

 Amount

If the offer is a set amount, enter the amount of the offer.

 Unspecified

If the offer is open to multiple payments, or the parent/carer decides how much to pay, select this.

If you want parents to see what they have paid so far you need to select the Share Balances box in Administration > Manage Accounts.

 Minimum

To ensure parents and carers can’t pay below an amount, enter the minimum amount here.

 Consent Required

Selected means parents and carers must give online consent for the offer.

 Required

Selected means that you need the parent or carer to make payment. Selecting this indicates within Joinos that this is important.

 Start Date

The date the offer becomes visible to parents/carers.

 End Date

The date the offer is removed and parents and carers can no longer see it. If you want the offer to continue indefinitely, select No end date.

 Recurring

Selected means that parents can pay more than once.

Min

The minimum number of items which can be purchased.

 Max

The maximum number of items that can be purchased in one transaction.

 Items Available

The number of items available in the offer.

Unlimited

You have an inexhaustible supply of items.

 Offer can only be purchased once

If this is a once only offer or you want the whole payment to be made for the offer, select this.

 Offer can be purchased multiple times or in instalments

If the offer will be repeated or you are willing to accept payment by instalments, select this.

 Total number of items you have

Offers can contain unlimited items or a fixed number, e.g. places on a trip are limited to the number of seats on a coach.

If there are a fixed number of seats or items in the offer, select Fixed total number of items and enter the number in the box alongside.

If the offer is not constrained by a limit, select Unlimited number of items.

 Cost of the items

Items can be of fixed cost or variable amount. For example, different sizes of uniform shirts may have different prices or one of an item might be more expensive and the cost decrease per item as the number of items increases.

Fixed
  1. Select Fixed cost and enter the fixed cost alongside.
  2. Enter how many items a parent can purchase at the same time.

    3. This will either be a fixed number or a variable number.

    • If the number is fixed, select Exactly and enter the number of items alongside.
    • If the number is variable, select Variable Quantity and enter the Minimum and Maximum number of items the parent can buy. This ensures they can only purchase a number that falls within these amounts.
 Variable
  • Select Variable Amount and enter the Minimum Amount the parent can pay alongside.
 Attachments

If you want to add attachments e.g. the trip itinerary or cost breakdown, you can add these to the offer and they will be displayed on the offer screen for parents/carers to view.

We recommend uploading PDF documents rather than Office files.

Do not upload any documents containing sensitive or private information.

  • To upload a new document, click Add a new file, enter a Title and Description for he file you are uploading, choose the file and click Upload.
  • To add an existing document, click Add a file you have uploaded previously and select the file you want to add to the offer from the list of files previously uploaded
 Names

Choose the names to add to the offer, by selecting names from the list of pupil names displayed. To help you shorten the list you can use the Filter or click Add Group to add a particular group of pupils e.g. a class or year group.

You can also edit the list by clicking Edit and then removing names from the list if you need to.

 Publishing

Selecting Post this offer makes the offer visible to parents on the start date you have chosen.

How do I ...

Create an offer?
  1. Select Account.
  2. Select VAT code if applicable.
  3. Select Ledger code.
  4. Select Settlement Account.
  5. Enter the offer Title and Description.
  6. Choose an icon for the offer if applicable.
  7. Select Gift Aid if you want to claim gift aid on the offer.
  8. Select Consent Required if you want parents to give online consent.
  9. Select Required if payment is required on the offer.
  10. Enter the Amount.
  11. Select Unspecified if the offer is open to multiple payments.
  12. Enter a Minimum amount.
  13. Enter the offer dates. If no end date is required, select the check box No end date.
  14. Select Recurring if required.
  15. Select whether the offer can be purchased once or multiple times. Select multiple times if you want to allow payment in instalments.
  16. Choose the offer quantities - Minimum, Maximum and Items Available.
  17. Select Unlimited if necessary.
  18. Enter the total number of items you have on offer. Select Fixed total number of items and enter the number alongside or select Unlimited number of items.
  19. Enter the cost of the item. If fixed cost, enter the fixed cost value and select how many items a parent can purchase at the same time. This might be an exact number of items or a quantity that falls within a range which you can specify by entering the minimum and maximum values. If a variable amount, enter the minimum amount that a parent can pay alongside.
  20. Add attachments to the offer if required.
  21. Select the names you want to make the offer available to.
  22. To save and post the offer, click Post Offer. To save the offer and work on it again later, click Save without Posting.
  23. Click Post this offer to make the offer visible to parents on the start date you have specified.
  24. Click Save to save the offer.

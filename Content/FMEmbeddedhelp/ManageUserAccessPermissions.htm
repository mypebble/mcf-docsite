Manage User Access Permissions

About the fields User

The user’s name.

 Edit Permission

Select Read only to ensure that the user does not have permission to edit. This means the user can’t edit the Transactions or Administration tabs.

If you only want to restrict access to either Transactions or Administration, then use the check boxes Admin and Transactions under Tabs and Functions instead.

 Tabs and Functions

Select check boxes for the restrictions you want to apply to the user, one at a time and save before managing the next user.

You can restrict a user’s access to Messaging, Admin, Transactions, Bank & Cash, Register, Offers, Reports and User Permissions.

If you want to restrict access to only Transactions and Administration, then select Read only under Edit Permission.

Use this screen to restrict access to parts of Fund Manager for different users.

Restrict user access?

You can change permissions for only one user at a time. So, for each user:

  1. Select the check boxes for the areas of the system you want to restrict access.
  2. Click Save.

Name Summary

About the fields Account

The user account taken items have been recorded against. For example, Dinners, Music Club, Family Burns Night Ceilidh.

 Group

The group of pupils who qualify for an item taken.

 Date From/To

The start date of the week the items were taken.

 Show Paid Balances

Click Search to display a list of all pupils in the Group and Class selected. alongside a grid for the week.

Summary

The unit cost for the item, which has been defined using Administration > Update Account.

 List of accounts

A list of pupils (who are in the selected group and class) is displayed along with the items taken during the week.

If a pupil is connected to a Joinos account, this is indicated in the J column.

Use this screen to see individual pupil balances.

You can also create Groups and Messages directly from the Names Summary Report now by clicking at the foot of the report screen. This is useful if you want to create a report or send a message to only those people who owe you money.

How do I ...

Find pupils?
  • Enter the Name you want to find (if you know it) in the Filter box.
  • Use the Filter to shorten the list of names that is displayed, by entering a character or characters you think are in the name.
  • If you aren’t sure if the name is out of use, you can show or hide out of use names in the following way:
    • Show out of use names?
      • Click Show out of use.

      Out of use names are shown greyed out in the List of Names
    • Hide out of use names?
      • Click Hide out of use.

      Only in use names are shown in the List of Names.
  • Use the paging controls to check for the name you want on another page.
Create groups?

Filter the list of accounts because creating a group will add all the names that are displayed in the list of accounts..

  1. Click Names.
  2. Click Create alongside Create group.
  3. Select the Group Type.
  4. Give the group a unique Group Name.
  5. Click Send to create the group.

    6. This group is available to use anywhere in Fund Manager now.

 Create an SMS message?

Filter the list of accounts because creating an SMS message will send the message to all the names that are displayed in the list of accounts.

You can only send SMS messages to people who have provided a contact phone number preferably a mobile number. Mobile numbers can be added to a student’s name, using Administration > Manage Names.

  1. Click Names.
  2. Click Create alongside Create message.
  3. Select the check box, Send SMS message.
  4. Enter the message which can be up to 160 characters.
  5. Click Send to send the message.
 Create a Joinos message?

Filter the list of accounts because creating a Joinos message will send the message to all the names displayed in the list of accounts which have a Joinos account.

  1. Click Names.
  2. Click Create alongside Create message.
  3. Select the check box Send Joinos message.
  4. Enter the Message title.
  5. Enter the Message body.
  6. You can add attachments:
    • To add attachment you have uploaded previously, click Add a file you uploaded previously and click Select alongside the attachment you want to add.
    • To add an attachment you haven’t uploaded, click Add a new file, enter a Name and a Description for the file, click Choose file and browse to select the file on your computer network. Then, when you have chosen the file, click Upload.
  7. Click Send to send the message.
 See the balance of a pupil’s account?
  1. Select the Account.
  2. Select the Group, if applicable.
  3. Select the Date From/To dates.
  4. To see all balances, including paid balances, select Show Paid Balances. If you only want to see outstanding balances, clear the Show Paid Balances check box.
  5. Click Search.

    6. The Receipts, Payments, Balance and Final Balance columns show the amount (£) for each pupil. The Paid, Free and Total columns are a count of the meals taken. The Free column next to Class indicates whether the pupil is eligible for Free School Meals.

 Identify who’s not paid and send them a message?
  1. Filter the list to display the pupils you are interested in e.g. by Group, Class, Year, etc.
  2. Clear Show Paid Balances.
  3. Click Names.
  4. Click Create Message.
  5. Enter the message details.
  6. Click Send.
 Print the balances?
  • Click Print.
 Export balances?
  • Export a list of balances to a CSV file by clicking Export CSV. This list is downloaded as name_summary.csv.
 View what has been taken in different ways?
  • To see what items have been taken, use Register > Taken Items.
  • To see what items a pupil has been recorded against, use Register > Statement.
  • To see the total number of free meals and universal free school meals, use Register > Universal Free School Meals.

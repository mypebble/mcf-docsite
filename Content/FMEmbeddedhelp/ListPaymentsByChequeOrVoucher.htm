List Payments by Cheque/Voucher Number

About the fields Include Cheque Transactions

Select to include cheque transactions.

 Include Cash Transactions

select to include cash transactions.

 From

Enter the start cheque number for cheque transactions or start voucher number for cash transactions.

 To

Enter the end cheque number for cheque transactions or end voucher number for cash transactions.

 List of Payments

The list of payments, displayed as a result of the criteria entered before clicking List Payments.

The list displays the payment Number, Bank Acc, Date, Type, Name, Description, Notes and Amount.

 Filter

Enter a character or characters in the transaction you want to find and the list of displayed payments adjusts accordingly to show a shorter list.

This report is particularly useful for reprinting payments that have already been entered.

How do I ...

Use the search criteria?
  1. Enter the criteria you want to display the results you want.
  2. Click List Payments.
 Change the display sequence of the payments?
  • Click the hyperlink column name to re-order the display of the payments. For example, clicking on Number will order the display in ascending or descending order. Clicking on Type. will order alphabetically. Clicking on Payment will order by amount.

You can’t re-order the list by Bank Acc or Name.

 Print an individual payment slip?
  1. Enter the relevant cheque and/or cash transaction number in the From and To fields.
  2. Clear Include Cheque Transactions or Include Cash Transactions if necessary.
  3. Click List Payments.
  4. Click Print.
 Print payments list as a table?
  • Click Print. A prompt tells you your report is being prepared. Once it has been prepared it is displayed as a PDF and you can then print the PDF file.

Manage Ledger Codes

About the fields Code

A code which is used to group transactions for a particular purpose.

 Bank Ledger Code

A code which is used to identify banks for reporting purposes.

You can only set up Bank Ledger Codes for Bank accounts that have already been set up using Administration > Manage Bank Accounts.

 Description

The name of the code.

The VAT and Cash in Hand Ledger Codes are set up by default and their Description cannot be changed. However, you can enter a Code name for these default codes so that they match the code names in your external financial system.

How do I ...

Create a code
  • Click New Ledger Code or Create New Bank Ledger Code and then follow the instructions on the Update Ledger Code screen.

You won’t be able to select a bank account for a bank ledger code on the Update Ledger Code screen unless you have already created the bank account, using Administration > Manage Bank Accounts.

 Amend a code
  • To amend the default ledger codes, VAT or Cash in Hand, click on the highlighted Description of the code.
  • To amend any other code, click on the highlighted Code that you want to update.

Then follow the instructions on the Update Ledger Code screen.

Register Account Summary

About the fields Account

The user account. For example, Dinners.

 Date From/To

The start and end date of the summary statement.

 Summary

The account summary shows the number of Groups assigned to the account, the amount in Receipts and in Payments as well as the Balance, the number of items which were Paid, Free and in Total.

 List of groups

A list of groups assigned to the account is displayed, showing the Group, Account, amount in Receipts, Payments and Balance, the number of Paid, Free and UIFSM items and the Total number of items.

Use this screen to see statements for the user accounts.

How do I ...

See a statement for an account?
  1. Select the Account.
  2. Select the Date From and Date To dates.
  3. Click Search.

    4. A summary is given of the number of Groups assigned to the account, the amount in Receipts and in Payments as well as the Balance for the account, the number of items which were Paid, Free and in Total.

    Below the summary a statement is shown for all groups assigned to the account.

Print the statement?
  • Click Print.
 Export summary?
  • Export the summary report to a CSV file by clicking Export CSV. This is downloaded as report.csv.

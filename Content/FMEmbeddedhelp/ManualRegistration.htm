sQuid Manual Registration

About the fields List of Names

A list of registered names is displayed, showing: Name, Class, Year, Reg, Groups, sQuid Card Number and CVV.

Use this screen to add or amend pupil’s sQuid information manually.

How do I ...

Search for pupils?
  • Use the Filter to shorten the list of names displayed in the list of registered pupils.
 Add or amend information manually?
  • Click the pupil Name to open the Set sQuid Card Number screen.

To search for pupils on the Set sQuid Card Number screen, use the Filter to shorten the list of names displayed in the list of registered pupils.

To add or amend information manually:

  1. Enter your sQuid Card Number for the pupil.
  2. Enter the card security CVV.
  3. Click Save.

About the fields ePayments File

The CSV file which contains the list of transactions that you want to import into Fund Manager.

 Upload

Uploads the CSV file to Fund Manager and displays the transactions to import at the foot of the screen.

 Transaction type

The type of transactions you want to import. You can select All transactions or import Cash, Cheque or ePayment transactions only.

 Description

A description of the transactions you are importing. Select a description from the drop-down list.

 Account

The Fund Manager Account to allocate the transactions against.

 Settlement account

This only applies if you are importing ePayment transactions. It is the account you will allocate the transactions against. Once you have imported you can transfer the total amount from the Settlement account into your desired bank account.

 List of transactions

The list of transactions contained in the import file, shows: the Date, Name, Class, Year, Description, Type, Amount and To Name.

You must match all transactions to import by selecting a To Name in Fund Manager.

Import files are CSV files which are created externally to Fund Manager. For example, within SchoolComms, SchoolMoney or ParentMail.

You can set up ParentPay , sQuid and Joinos so that transactions are automatically imported from those packages. For information on how to do this, see Import ParentPay Transactions Automatically, Using sQuid with Fund Manager and the Joinos User Guide.

How do I ...

Import Transactions?
  1. Select the CSV file to import.
  2. Click Upload.
  3. Click Filter to filter the list of transactions displayed according to the type or description that you want to import if you have multiple account codes to import.
  4. Select the Fund Manager Account to allocate the transactions against.
  5. If you are importing ePayment transactions, the Settlement Account should be selected automatically for you to allocate the transactions against.

    6. If you don’t have a settlement account to select, please contact Pebble Support to create one for you.

  6. Work through the list of transactions, selecting/checking and matching To Name from the drop-down list of names

    7. You should only import one page at a time, so ensure you click Import when you match all the transactions to the To Name on each page.

    The closest matching name appears at the top of the drop-down list. If the correct name is not available for you to select, you may need to create a new name, via Administration > Manage Names > Create New Name and follow the instructions on the Create New Pupil or Create New Organisation tab.

  7. Click Import to import the transactions on the displayed page of transactions.

    8. When you successfully import a page of transactions, a View Pending Import button appears alongside the Upload button.

    You can leave the screen and return later. If you do, don’t upload the file again. instead click on View Pending Import and carry on from the point you left the screen.

  Continue an import which was interrupted

If you have navigated away from the screen after importing some transactions, don’t click Upload File to import the entire file again, instead click View Pending Import and carry on from where you had left off before.

 Transfer total amount from settlement account
  • Go to Bank & Cash > Transfer between Bank Accounts and follow the instructions on the screen there.

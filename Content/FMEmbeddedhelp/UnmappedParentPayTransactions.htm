Unmapped ParentPay Transactions

About the fields Transaction List

A list of transactions which are unmapped is displayed, showing: transaction Date, Account, Settlement, ParentPay Name, name Matched, Description and Amount. The most recent transactions are shown at the top of the list.

Use this screen to view transactions that have not or do not need to be mapped.

If you identify a payment item you can’t see in Fund Manager, this might be because you haven’t mapped a name and or an account to this transaction type. If so, you need to Map Names and Payment Items.

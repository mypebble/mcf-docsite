About the fields List of Names

The list of pupil names is displayed, showing: Name, Joinos Code, code Creation Date, code Expiry Date, Class, Year, Reg and Letter.

Use this screen to see the Joinos codes and expiry dates for pupils and staff who have had a Joinos code created for them with a Creation and Expiry Date and who have not yet been activated in Joinos. From this screen you can print letters for pupils and staff which tell them how to log on and activate their account.

Codes are valid for three months, so if they don’t activate their account within that time you will need to recreate their code on the Create Joinos Codes screen.

How do I ...

Print Joinos letters for groups of pupils?
  1. Click Print Joinos Letters.
  2. On the Print Joinos Letters screen, add groups to print letters for and click Generate.

    3. You receive an email when the letters have been created for the groups you have selected.

    You can access these on the Joinos Letters tab.

 Print a Joinos letter for an individual pupil?
  • Click Print in the Letter column for the pupil.

    • The letter is prepared and displayed in a new browser window for you to download or print.

 Export the Joinos codes?
  • Click Export CSV. A CSV file is downloaded automatically as report.csv. You can then use this report in your own Joinos template letter.

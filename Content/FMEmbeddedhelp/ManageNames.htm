 List of Names

The list of user names that have been created, is displayed. The list displays the following information for each name: Name, Class, Year, Registration number, Groups and whether the pupil is a gift aid donor GA.

From here you can create a new name or organisation by clicking Create New Name or Create New Organisation.

 Show out of use / Hide out of use

Show out of use will include all names marked as Out of use in the list of names displayed. If a name is out of use, it is greyed out in the list so that you can easily recognise that it is not in use.

Hide out of use only shows names in the list which are marked as In use.

 Batch Actions

You can select batches of names in the list and mark them as In Use or Out of Use with one click of Update Name(s).

You can make it easy to apply batch actions to names by creating groups. See Group Actions.

 Group Actions

You can create groups to assist you in selecting names quickly when you want to add transactions for the names in the group, or include the names in the group in reports.

Groups are easily managed from the Group Actions section of the Manage Names screen.

Names can be assigned to more than one group, so you can design the groups that will make your workday easier.

Add and manage pupil, teacher and staff names and organisation names within Fund Manager. This lets you keep a track of all names you need in Fund Manager to record transactions correctly.

How do I ...

Create Names?

You can import pupil names from your administration database. You can also create names individually, if required, using Create New Name.

Import Names

You can import pupil names if these are prepared as a CSV file.

We recommend that you import names from your administration database. This should prevent duplication of records within Fund Manager. The MISAPP feature is the automatic way to add an up-to-date file of pupil names with their year and registration group details and import them quickly into Fund Manager. For more information, see MISAPP Preparation Instructions on the Support website.

 Create a new name individually
  • Click Create New Name and follow the instructions on the Create New Pupil screen.
 Find Names?
  • Enter the Name you want to find (if you know it) in the Filter box.
  • Use the Filter to shorten the list of names that is displayed, by entering a character or characters you think are in the name.
  • Re-order the list by clicking the hyperlinked column name, so if you know that the name you want is in a year group then order by Year, to group all names in the year together.
  • To see a list of only the names in a class, year, registration group or group, click the hyperlinked Class, Year, Reg or Group.
  • If you aren’t sure if the name is out of use, you can show or hide out of use names in the following way:
    • Show out of use names?
      • Click Show out of use.

      Out of use names are shown greyed out in the List of Names
    • Hide out of use names?
      • Click Hide out of use.

      Only in use names are shown in the List of Names.
Edit name details
  • Click the hyperlinked Name and follow the instructions on the Update Pupil screen.
 Manage Groups?

Groups assist you in selecting names quickly when you want to assign transactions to the names in the group, or include the names in the group in reports.

Groups are easily managed from the Group Actions section of the Manage Names screen.

Names can be assigned to more than one group, so you can design the groups that will make your workday easier.

Create Group

Add a new group by clicking Create Group. The Manage Groups screen appears.

  1. Enter a name for the group.
  2. Select a Group Type from the list of available group types.

    3. There are some group types provided by default for you to choose, for example, School, Class, Year, Registration

    If you create a group for your Class 1A, you would select Class for the Group Type. For any group you create which does not fit into one of these types, for example, a swimming group which has pupil members from different years or classes, you would select the Group TypeUser Defined.
  3. Click Add Group. Once you have done this you will be able to select it from the Select Group drop-down list.
Select Group

Select a group you have already created from the drop-down list of groups available.

 Add to Group
  1. Select the names in the List of Names that you want to add to a group.
  2. Select the group you want to add the names to from the Select Group drop-down list.
  3. Click Add to Group.

    4. The group name is added to the Groups column for the selected pupils.

 Remove from Group
  1. Select the names in the List of Names that you want to remove from the group.
  2. Select the group from the Select Group drop-down list.
  3. Click Remove from Group.

    4. The group name is removed from the Groups column for the selected pupils.

 Delete Group

You can only delete groups which don’t have any names assigned.

  1. Select the group you want to delete from the Select Group drop-down list.
  2. Click Delete Group.
  3. At the prompt to confirm that you really want to delete the group, click Delete.
 Use Batch Actions?

The batch actions available are to mark names In Use or Out of Use.

  1. Select the names in the List of Names that you want to apply the batch actions to.
    • If you have created a Group for these names you can select the Group name.
    • If you want to select out of use items, ensure that you click Show out of use to display these names.
  2. Select In Use to mark all the selected names as in use, or Out of Use to mark all the selected names as out of use.
  3. Click Update Names to apply the action to the names.

    4. When you have applied the action, clear the selection from the name before applying further changes. It isn’t removed automatically.

 Consolidate names?

You have found out that you have two records for the same name and want to create one record instead of two.

  1. Locate the name in the List of Names which you want to retain as the record for the name.
  2. Click the hyperlinked Name to open the Update Pupil screen, click the Consolidate Pupil tab and follow the instructions on the Consolidate Pupil screen.

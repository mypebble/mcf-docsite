Manage Names

A short-cut to launch the Administration > Manage Names screen.

Use this screen to enter and send SMS text message.

You can only send SMS messages to people who have provided a contact phone number, preferably a mobile number. Mobile numbers can be added to a student’s name, using Administration > Manage Names.

How do I ...

Send an SMS message?
  1. Enter the Text that you want to send.
  2. Select the Contacts that you want to send the message to.
  3. Click Send.
 Add a contact to the list of contacts?
  • Click Manage Names, add a phone number for the contact

sQuid View Registered Names

About the fields Class

The pupil’s class.

 Year

The pupil’s year.

 Reg Group

The pupil’s registration group.

 Search

Click to find and display registered names by class, year or registration group, or a combination of these.

 List of Names

A list of registered names, showing: Name, Class, Year, Reg, Groups, sQuid Card Number and CVV.

Use this screen to see who is registered with sQuid and all cards that are registered to the pupil (old, lost, expired and so on).

Fund Manager knows which card is linked to sQuid and uses the correct number.

How do I ...

Search for pupils?
  • Use the Search by fields to list pupils by class, year or registration group.
    • Select the Class, Year and / or Reg Group.
    • Click Search.
  • Use the Filter to shorten the list of names displayed in the list of registered pupils.
 Print registration letters?
  • Select the pupil names you want to print registration letters for.
  • Check the card has the correct purses added.
  • Click Print Registration Letters.
 Export list of registered pupils?
  • Export a list of the registered pupils to a CSV file by clicking Download CSV. The list of registered pupils is automatically saved to your downloads as a report.CSV file.

View Statement

About the fields List of reconciled accounts

A list of reconciled bank accounts is displayed, showing the Bank Account, Opening Balance, Balance, Start Date and End Date.

This report lets you access and view previous reconciliation statements for different bank accounts. It is useful when you are performing a period end because you can check if all your bank accounts are reconciled up-to-date.

How do I ...

View previously reconciled bank statements?
  • Click the Start Date or End Date of the statement you want to view. This opens the statement file for the bank account, displaying the transactions that were reconciled during the statement period.
 Print the statement?
  1. Click the Start Date or End Date of the statement you want to view. This opens the statement file for the bank account.
  2. Click Print. A prompt tells you your report is being prepared. Once it has been prepared it is displayed as a PDF and you can then print the PDF file.

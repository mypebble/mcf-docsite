About the fields Contents

The message Title and Body is shown at the top of the screen.

 Filter

Enter a character or characters you want to find and the list of displayed names adjusts accordingly to show a shorter list to choose from.

The filter works by displaying all names which contain the character or characters you have entered.

 List of Names

A list of names that the message has been sent to.

Use this screen to review the message that was sent, who it was sent to and whether they have read the message or not.

How do I ...

Check if a recipient has read my message?
  • Use the Filter to find the recipient and check the Read column to see if they have read the message or not.

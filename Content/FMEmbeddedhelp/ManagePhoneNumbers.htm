Manage Phone Numbers

About the fields List of Names

A list of names is displayed with the following information for each name: Class, Year, Registration number, Groups the name is assigned to, and the Primary Phone number.

You can send SMS text messages from within Fund Manager, using Messaging > Send Text Message. There are also other places within Fund Manager where you can do this, for example, you can save and send an SMS when you enter a receipt or a payment. SMS messages can only be sent to a Primary Phone number which you must select for a name here.

How do I ...

Change the display sequence of the names?
  • Click the hyperlink column name to re-order the display. For example, clicking on Name will order the display alphabetically. Clicking on Class. will order the display by class.
Add, edit or remove a phone number?
  • Click the hyperlinked name to display the Update Pupil screen and follow the instructions on that screen.
 Select a Primary phone number?
  • Select a phone number from the Primary Phone drop-down list.

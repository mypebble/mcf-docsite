List Receipts by Receipt Number

About the fields From

Enter the start receipt number.

 To

Enter the end receipt number.

 Include

Select the types of payment.

 List Receipts

Click List Receipts to display a list of receipts that fit the criteria entered.

 List of Receipts

The list of receipts, displayed as a result of the criteria entered before clicking List Receipts.

The list displays the receipt Account, Number, Date, Type, Name, Class, Description, Notes and Amount.

 Filter

Enter a character or characters in the transaction you want to find and the list of displayed receipts adjusts accordingly to show a shorter list.

This report is particularly useful for reprinting receipts that have already been entered.

How do I ...

Use the search criteria?
  1. Enter the criteria you want to display the results you want.
  2. Click List Receipts.
 Change the display sequence of the receipts?
  • Click the hyperlinked column name to re-order the display of the receipts. For example, clicking on Number will order the display in ascending or descending order. Clicking on Type. will order alphabetically. Clicking on Amount will order by amount.

You can’t re-order the list by Name or by Class.

 Print receipts list?
  • Click Print. A prompt tells you your report is being prepared. Once it has been prepared it is displayed as a PDF and you can then print the PDF file.
 Reprint Receipts?
  • Reprint the receipts by clicking Reprint Receipts.

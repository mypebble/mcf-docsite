sQuid Active Offers

List of Offers

A list of active offers is displayed, showing: offer Title, Code, Account, Start Date, Deadline and End Date.

 Filter

Enter a character or characters in the offer you want to find and the list of displayed offers adjusts accordingly to show a shorter list to choose from.

The filter works by displaying all names which contain the character or characters you have entered.

Use this screen to see what offers are currently active and to amend an offer if required.

How do I ...

Find Names?
  • Enter the Title you want to find (if you know it) in the Filter box.
 Amend an offer?
  • Select an offer Title to display the offer details and then amend as required.
 Remove an offer?

Select an offer Title to display the offer details and then change the current End Date to before the Start Date. The offer will be removed from Fund Manager and sQuid.

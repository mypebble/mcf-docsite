Map Payment Items

About the fields List of Payment Items

A list of payment items is displayed, showing: Name, Group 1, CRF Code, Account, Hide Payment Items. The ParentPay payment item Name is shown on the left hand side of the screen. Fund Manager Account names are to be matched and selected on the right.

Use this screen to map accounts in Fund Manager with your Payment Items accounts in ParentPay to ensure that the transactions import into the correct names and accounts in Fund Manager.

You will need to map an account in Fund Manager when you create a payment item in ParentPay which has transactions that you need to import into Fund Manager. However, you will only see the new payment item in Fund Manager after the first transaction is processed.

Some accounts might not need to be mapped as the money may be going to a different bank account and so these transactions in ParentPay don’t need to be imported into Fund Manager at all; you can use the Hide button to archive these payment items.

How do I ...

Match payment items?
  1. Select the Fund Manager Account name to match the ParentPay account Name, ignoring any accounts you don’t want to import transactions from into Fund Manager. You can hide these unwanted payment items, using the Hide button.
  2. Click Map Payment Items.

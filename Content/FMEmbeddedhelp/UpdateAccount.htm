About the fields Account Code

A unique alphanumeric code which identifies the account. It can be up to six characters long.

 Account Name

The name of the account, which describes its purpose, e.g. School Book Fair

 In Use

Selected indicates that the account is in use.

 Share Balances

If selected, then you can share the balance through Joinos so parents can see what is left to pay on an offer.

You will need to create the offer as usual within Joinos, using Offers > Create Offer, then select the Unspecified check box and enter the minimum amount you would like parents to pay, complete the offer with relevant attachments, add names to the offer and then post to make it visible to parents. Once a parent has paid the deposit, they will be able to see what they have left to pay, via the Balances tab in Joinos.

 Register Account

If selected, this marks the user account as a Register Account and you can then use the Register tab to record attendance for various items on the account e.g. meals, clubs, events and so on.

You also need to enter the following details for a Register Account:

Exclusion Group

This is a group that would not be expected to pay for the item you are recording attendance for in the Register tab.

You must define an exclusion group even if you don’t assign any pupils to it as one exclusion group is required within Fund Manager.

For example, you might create an exclusion group called Free Trips and add any pupils to the group who would not be expected to pay for the item in the usual way from Administration > Manage Names.

If you don’t have an exclusion group set up already, then click Manage Groups which opens the Manage Groups window where you can add one.

Display on Device

If selected, you can view and record attendance on an Android Tablet device, provided you have purchased the device direct from us.

If you have purchased a Tablet elsewhere you can contact Pebble Support to check whether your tablet is compatible with the Register App.

 Attendance Group

The group which is used for recording pupil attendance, including pupil meals for those who don’t qualify for free meals or for Universal Infant Free School Meals.

 UIFSM Group

This only applies to state-funded pupils in England in Reception, Year 1 and Year 2 classes.

The group which is used for recording Universal Free School Meals.

What if I don’t have a UIFSM group already created?

If you don’t have a group for Universal Infant Free School Meals, you can create one using Administration > Manage Names > Create Group. As well as giving the group a name, you must select the Group Type, user-defined.

Then you must assign the relevant pupils to the group (currently this is all Reception, Year 1 and Year 2 pupils in state-funded schools in England).

 Opening Balance

The amount of money that the account has on the date the account was created.

 Budget

A positive or a negative value which indicates an expected level of income or profit (positive) or expenditure or loss (negative) on the account.

Unit Cost

A cost per pupil or per head for an activity or trip.

If you want to use the report, Who’s not paid, then you must enter a Unit Cost here.

 Groups Belongs to
  • Add the account to an existing group by clicking on the group name in the left hand window to move it to the right hand window.
  • Remove the account from an existing group by clicking on the group name in the right hand window to move it to the left hand window.
 New Group

Enter the name of a new group, if required, in the New Group field. The new group will be created and the account added to the group when you click Save.

Use this screen to update user accounts. These are equivalent to individual columns on a manual ledger and help you to report on income and expenditure for different items, e.g events, locations or classes. There is no limit on the number of accounts you can have.

How do I ...

Amend an account?
  1. Enter a unique Account Code.
  2. Enter an Account Name.
  3. Select In Use to ensure you can select the account on the system.
  4. If you want to share the balance through Joinos so parents can see what’s left to pay on an offer, select Share Balances.

    5. You will need to create the offer as usual within Joinos, using Offers > Create Offer, then select the Unspecified check box and enter the minimum amount you would like parents to pay, complete the offer with relevant attachments, add names to the offer and then post to make it visible to parents. Once a parent has paid the deposit, they will be able to see what they have left to pay, via the Balances tab in Joinos.

  5. If you want to register pupil attendance and take-up of meals, using the Register tab within Fund Manager, then select Register Account.
    1. Select an Exclusion Group for the account.

      2. You must select an Exclusion Group even if you don’t have any pupils assigned to it.

    2. If you have an Android Tablet you have purchased direct from Pebble, and you want to view and record attendance on this, select Display on Device.

      3. If you have any other Tablet you must check with Pebble Support for its compatibility.

  6. Amend the Opening Balance for the account, if required.

    7. For example, if you have discovered that your user accounts and bank accounts don’t balance, and have identified the reason why you can increase or decrease the user account opening balance so that the user accounts and bank accounts balance correctly.

    Don’t use this option as a short-cut to entering a transaction on the system to get your accounts to balance. Entering a transaction - either anew if you had not entered it before, or as a contra-entry to cancel a mistaken entry - is the best way to get your accounts to balance as this provides traceability when you come to audit your accounts.

    If you amend the user account opening balance so that your user accounts and bank accounts don’t balance you will not be able to complete your period end closedown.

    If you need to amend the opening balance, click Amend Opening Balance.

    To increase the opening balance:

    1. Select Increase.
    2. Enter the value by which you want to increase the opening balance. For example, to increase by £10, enter 10.00 in the balance by box.
    3. Click Set Opening Balance Change.

    To decrease the opening balance:

    1. Select Decrease.
    2. Enter the value by which you want to decrease the opening balance. For example, to decrease by £25.30, enter 25.30 in the balance by box.
    3. Click Set Opening Balance Change.
  7. Enter a Budget value for the account.
  8. If you want to use the report, Who’s not paid, enter a Unit Cost (cost per pupil head).
  9. Select which Groups the account Belongs to by clicking the group name in the left-hand window. Or if you need to create a new group, enter the name in the New Group field.
  10. Click Save.
 Remove account details?
  • Clear the In Use check box and click Save.
 Add an account?
  • Use Administration > Manage Accounts > Create New Account, and follow the instructions on the Create New Account screen.

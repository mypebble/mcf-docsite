Update Bank Account

About the fields Code

The bank account code, e.g. CURR for Current Account.

 Opening Balance

The value of the account when you create it.

The Opening Balances of your accounts are usually entered by one of the Pebble Support team when you first begin using Fund manager but you can amend this if it has changed since the account was set up for you. You can also enter opening balances for any accounts that you create yourself.

 Name

The bank account name, e.g. Current Account.

 In Use

Selected means that you can use the bank account. If you no longer need to use a bank account, you can take it out of use by clearing this check box.

 Default Account

The bank account that you use most frequently, e.g. the Current Account.

Use this screen to enter details for a bank account.

The current account with code CURR and name Current Account is created for you by default. This is the most frequently used account in most schools. You can change the account name of this account but you should leave the code as CURR to ensure that it stays at the top of the alphabetical list of bank accounts.

How do I ...

Amend a bank account?
  1. Amend the details as required.
  2. Click Save.
 Set an account up for electronic bank payments?
  • Please contact Pebble Support.
 Mark an account as the default account?
  • Select the Default Account check box.
 Remove a bank account?
  • Clear the In Use flag.
 Create a new bank account?
  • Use Manage Bank Accounts > Create Bank Account to display a blank Update Account screen and fill in the screen, following the instructions above.

How do I ...

Amend user details?
  1. Enter the user Email Address, First Name and Last Name.
  2. Ensure In Use is selected, if the user is still using Fund Manager.
  3. Click Save.
 Remove user details?
  • Clear the In Use check box and click Save.
 Add a new user?
  • Use Administration > Manage Users, select UserName and follow the instructions on the Create New User screen.

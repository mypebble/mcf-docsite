Set Ledger Codes

About the fields Date from

The date from which to display transactions.

 Date to

The date to which to display transactions.

 Search

Clicking Search will display transactions between the Date from and Date to dates.

 List of transactions

A list of transactions is displayed with the following information for each transaction: transaction date, user account code, user account name, transaction reference, receipt amount, payment amount, current ledger code and new ledger code.

 New Ledger Code

The new ledger code for the account.

You can’t amend transactions that have been paid into the bank, reconciled or included in the VAT 100 report and so these transactions won’t be included in the list of transactions displayed.

How do I ...

Display transactions?
  1. Use the Date Range fields to select the transactions to display.
  2. Click Search to display the transactions you can amend which fall within the date range.
 Set the ledger codes for receipts or payments?

Only applicable to transactions in the current period which have not been paid into the bank, reconciled or included in the VAT 100 report.

  1. For each transaction displayed that you want to set a new ledger code for, select the New Ledger Code from the drop-down list of ledger codes available.
  2. When you have finished click Save.
 Amend a receipt or payment

Only applicable to transactions in the current period which have not been paid into the bank, reconciled or included in the VAT 100 report.

  • Select Amend Receipt/Payment and follow the instructions on the Amend Receipt/Payment screen.
 Set a new tax code for transactions?

Only applicable to transactions in the current period which have not been paid into the bank, reconciled or included in the VAT 100 report.

  • Select Set VAT Status and follow the instructions on the Set VAT Status screen.

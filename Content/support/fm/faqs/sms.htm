How do I use SMS Messaging?

Issue: I need to send a message to certain parents to bring an important item to their attention.

Solution: SMS messages can only be sent to those Parents/Carers who have provided you a contact phone number preferably a mobile. (This facility is automatically available only with *MISapp).

A mobile number can be added manually to a student’s name using the Administration > Manage Names function.

To send an SMS go to Messaging > Send Text Message.

Type in the text message that you want to send

A list appears of all the students, select those students you wish to send the message to, and click Send

Integrating with PS Financials

The process for integrating with PS Financials is simple and involves initial setup in Fund Manager and subsequent export of transactions from Fund Manager and import into PS Financials which is handled automatically. You can also export transactions from Fund Manager manually and import these into PS Financials manually, if required.

Initial setup in Fund Manager

In order to integrate Fund Manager with PS Financials, you need to set up Fund Manager so that the bank and ledger codes in Fund Manager match the codes you already have in PS Financials.

Set up bank account codes

  1. Open Fund Manager > Administration > Manage Bank Accounts.
  2. Click Create Bank Account and create your cash account with the same Code as the cash account in PS Financials.
  3. Click Create Bank Account and create your current account with the same Code as the current account in PS Financials.

Set up VAT and cash in hand ledger codes

  1. Open Fund Manager > Administration > Manage Ledger Codes.
  2. Click VAT to display the Update Ledger Code screen.
  3. Enter the nominal part of the VAT Ledger Code in PS Financials as the Fund Manager Code and click Save.
  4. Click Cash in Hand to display the Update Ledger Code screen.
  5. Enter the nominal part of the Cash in Hand Ledger Code in PS Financials as the Fund Manager Code and click Save.

Set up account codes

  1. Open Fund Manager > Administration > Manage Accounts.
  2. For each ledger in your PS Financials Ledger List, create a new account.
    • Click Create New Account.
    • Enter the nominal part of the Ledger code in PS Financials Nominal Maintenance as the Fund Manager Account Code and click Save.

Automatic synchronisation setup

This is set up using the PSF Integration app and must be done by the school IT administrator. The integration app should be installed on the school IT server which is not switched off. This ensures automatic integration takes place overnight.

The date of automatically imported transactions is set during the import to the date of import. Document Type must be specified for import.

Manual export from Fund Manager and Manual Import into PS Financials

Export from Fund Manager

  1. Open Fund Manager > Reports > Account Summary.
  2. Click Export to PS Financials.

    3. This outputs a CSV file which contains:

    Nominal Code This is set in Fund Manager > Manage Ledger Codes to match the nominal part of the ledger code in PS Financials.
    Account Code This is set in Fund Manager > Manage Accounts to match the nominal part of the ledger code in PS Financials Nominal Maintenance.
    Date This is automatically set as the date of export.
    Period This is calculated from the date on the transaction.
    Description This comes from the transaction in Fund Manager.
    Amount This is set in Fund Manager.
    Tax Code This comes from the transaction in Fund Manager.

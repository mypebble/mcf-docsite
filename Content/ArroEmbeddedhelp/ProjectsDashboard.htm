Projects Dashboard

About the fields Active

A list of your current active projects.

When you create a project it is automatically listed in the active projects list.

Inactive

A list of your current inactive projects.

These are projects which you do not want to work on currently or have already completed.

 Project Name

The project title, entered when you create the project.

 Funding Target

The amount of money required for the project. Two amounts are shown: the first is the amount of money that has been raised to date, while the second is the total amount of money to raise.

 Completion Date

The date when the project must be complete. If there is no completion date it is marked as Ongoing.

 Project Strength

A percentage value which indicates how complete the project is in terms of its narrative and description.

 Comment

A number indicates the amount of comments that have been received on the project.

 General Pot

The general pot is where all your income goes if it hasn't been allocated to a project. You can use the general pot for activities that can be used to fund multiple projects, such as open donations. Use the transfer facility to allocate money to a project from the general fund.

 New Project

When you want to create a new project, click New Project.

 Mark Inactive

When you have completed a project or if you want to cease work on it, you can mark it as inactive by selecting the project name in the list of projects and clicking Mark Inactive.

Use this screen to manage your projects, create and edit projects, and mark them as inactive if they are not required.

Active projects display on the Active tab and Inactive projects display on the Inactive tab. New projects are marked as Active by default.

You can only edit projects which are active, so in order to edit a project, it must first be marked as Active.

How do I ...

Create a project?
  • Click + New Project.

    • You can base your new project on a project template or create from scratch by clicking Create Custom, from the Choose a Project Template screen which appears.

 Edit a project?
  • On the Active tab, click the Project Name to display the project details, then click Edit Page to edit the details.
 Mark an active project as inactive?
  1. On the Active tab, select the check box before the Project Name.
  2. Click Mark Inactive.
 Mark an inactive project as active?
  1. On the Inactive tab, select the check box before the Project Name.
  2. Click Mark Inactive.

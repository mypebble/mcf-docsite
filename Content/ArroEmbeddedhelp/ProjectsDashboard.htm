Fields Active Projects

A list of current project names.

 Funding Target

The amount of money required for the project to be successful. Two amounts are shown: the first is the amount of money that has been raised to date, while the second is the total amount of money to raise.

 Completion Date

The date when the project must be complete. If there is no completion date it is marked as Ongoing.

 Project Strength

A percentage value which indicates how complete the project is in terms of the narrative and description.

Comments

A number indicates the comments which have been received on a project.

About the dashboard

This is where you create and manage your projects.

  • To add a new project, click Add New.
  • To edit an existing project, click the Active Project name.
  • To see how much income you've not yet allocated to a project, click General Pot.

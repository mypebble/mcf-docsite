About the fields Filters

Use the filters at the top of the screen to shorten the list of project templates to choose from. You can filter the list on category, cost, impact and timescale.

 View Template

Clicking View Template displays the details of the template so you can assess whether you want to base your project on the template or not. The details include a description of the project, why you need your project and what impact your project will have.

These details tell the story of what you are trying to achieve, emphasising the gains the school and the local community will make through the project.

 Start Date

The date the project will start or has already started.

 Completion Date

The date you expect the project to be completed, if you have a date.

 No Completion Date

If the project does not have a completion date, select No Completion Date.

 Enter your Target Amount

The amount of money which you need to raise for the project.

Use this screen to choose a project template to base your new project upon or choose to create a custom project from scratch.

There are a number of standard templates provided in Arro for you to choose from. Over time this number will increase. The aim is to help you create your project narrative and set up your project more quickly and easily than from scratch. You may choose to create a project from scratch but it is often helpful to create one from a template and then amend the details to tailor it to match your own project.

How do I ...

Choose a template?
  1. Use the filters at the top of the screen to display the types of template which most closely match your project.

    2. For example, select Fitness/Sports from the Category drop-down if your project is around facilities for sports. You can also filter by Cost, Impact and Timescale.

  2. Use the number keys at the foot of the page to scroll through the templates listed.
  3. Click View Template to read the details of the template. These include the project description, why you need the project and what impact your project will have.
  4. If you want to use a template, scroll to the foot of the template and enter the following details:
    1. Start Date for the project.
    2. Completion Date for the project, if you know this. Alternatively, select No Completion date.
    3. Enter your Target Amount.
  5. Click Choose this Template.

    6. The new project is created for you based on the template and you can now edit it, following the help on the view or edit project screen.

 Create a project from scratch?
  • Click Create Custom and follow the instructions on the Create Project screen.

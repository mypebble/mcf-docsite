Use this screen to choose a project template to base your new project upon or to choose to create a custom project from scratch.

There are a number of standard templates provided in Arro for you to choose from. Over time this number will increase. The aim is to help you create your project narrative and set up your project more quickly and easily than from scratch. You may choose to create a project from scratch but it is often helpful to create one from a template and then amend the details to tailor it to match your own project.
